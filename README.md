------------
Installation
------------
Engine can be used with any UCI-compliant GUI (Graphical User Interface)
* [Chess Arena](http://www.playwitharena.de/)
* [The Tarrasch Chess GUI](https://www.triplehappy.com/)
* Chessbase / Fritz
* Chess Assistant
* Chess Partner
